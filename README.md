# CP2 - Java Advanced - Spring Security + JWT

## Instruções
Criar uma API nova para o contexto de Diploma com as classes e atributos listados abaixo. 
Essa API deve configurar Spring Security com JWT, liberando acesso ao USER apenas no GET de Diploma, que deve retornar um texto específico listado abaixo, 
liberando ao ADMIN todos endpoints, deixando os endpoints de registro e login de usuários liberado para todos. 
Importante: não deve ser feito no projeto atual nem usar a mesma secret key que utilizamos no exemplo. 
O projeto também deve contemplar Validation para que todos os campos do texto gerado estejam corretos dentro do contexto. 
A entrega deve ser no Github. Apenas um do grupo deve subir o link aqui na tarefa. Os nomes dos integrantes devem estar no README do projeto (um projeto apenas por grupo!).
